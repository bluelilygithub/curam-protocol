## Quality Checklist for New Industry Pages

Use this checklist when creating any new industry page:

### Content Quality
- [ ] Headline addresses emotional pain point (not generic "AI Automation")
- [ ] "The [Industry] Trap" framework clearly defines market squeeze
- [ ] "What If You Recovered X%" section includes transparent math
- [ ] 3 concrete automation examples (not vague "intelligence")
- [ ] Industry-specific compliance/governance section
- [ ] 4-point feature grid (Compliance, Security, Training, ROI)
- [ ] Social proof metrics (4 specific numbers)
- [ ] Case study reference with link
- [ ] 5-7 industry-specific FAQs
- [ ] Dual CTA (Feasibility Sprint + ROI Calculator)

### Technical Accuracy
- [ ] Numbers are realistic (not aspirational)
- [ ] Integration examples name actual software
- [ ] Compliance references cite actual regulations
- [ ] Case study metrics are verifiable
- [ ] ROI calculator link includes `?industry=` parameter

### Tone & Voice
- [ ] Professional but conversational (not corporate jargon)
- [ ] Addresses "you" directly (not "firms" or "organizations")
- [ ] Specific > Generic (always)
- [ ] Quantified > Vague (always)
- [ ] Credibility > Hype (always)

### SEO & Conversion
- [ ] Primary keyword in H1 headline
- [ ] Industry name in URL slug
- [ ] Meta description under 160 chars
- [ ] 2-3 internal links to Protocol phases
- [ ] Link to relevant case study
- [ ] Schema markup for LocalBusiness (if applicable)

## Summary: Best Practices

### What Makes a Great Industry Page?

1. **Specificity Wins Trust**
   - "48 hours/week √ó $250/hr √ó 48 weeks = $576,000" > "Significant savings"
   - "ActionStep, LEAP, Smokeball" > "Leading practice management systems"
   - "90%+ accuracy" > "High accuracy"

2. **Show Your Work**
   - Transparent math (10% Capacity Shift section)
   - Concrete automation examples (3-way match, GL coding)
   - Step-by-step timelines (48-hour feasibility sprint)

3. **Address the Elephant**
   - FAQs that tackle real objections (privilege, errors, ROI timing)
   - Compliance/governance section (data sovereignty, audit trails)
   - Risk-free testing emphasis ($1,500 refund guarantee)

4. **Industry-Specific = Credibility**
   - Name their software (not generic "systems")
   - Cite their regulations (TPB, ATO, Law Society)
   - Use their language (trust accounts, matter files, GL codes)

### What Kills Conversion?

1. ‚ùå **Vague promises** - "AI-powered automation" (what does that mean?)
2. ‚ùå **Missing FAQs** - Leaves objections unaddressed
3. ‚ùå **Generic examples** - "Document processing" (show the actual workflow!)
4. ‚ùå **Aspirational ROI** - "Up to 80% savings" (show realistic math)
5. ‚ùå **No exit strategy** - "Long-term partnership" (Phase 1 should have escape hatch)

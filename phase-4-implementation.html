Questions About Phase 4?

Industry-specific answers to implementation concerns

For Accounting Firms: What happens if the AI misreads a GL code and posts to the wrong account?

+

Wave 1 includes a "Human-in-Loop" flag for low-confidence extractions. Any GL code below 95% confidence gets queued for review before posting. You control the threshold. Most firms start conservative (98%) and relax once trust is established.

For Accounting Firms: Can we test on real invoices without posting them live?

+

Yes. The first 3 days of Wave 1 Go-Live run in "read-only" mode. All extractions are validated against your manual entries. No postings until you approve.

For Engineering Firms: Will this work with our existing CAD/BIM system?

+

Phase 2 would have identified integration requirements. We support Revit, AutoCAD, BricsCAD, and most PDF-output systems. If your CAD exports PDFs, we can extract from them. For direct BIM integration (IFC files), we scope that in Phase 2.

For Engineering Firms: Can you handle redlined/markup drawings?

+

Yes, but accuracy drops to ~85% (vs 95% on clean drawings). Wave 2 typically includes a "Redline Detection" workflow that flags these for manual review. The AI learns common markup patterns over time.

For Logistics Companies: How fast is this? We process 1000+ documents per day.

+

Processing speed is 5-10 seconds per document. For 1000 docs/day, that's ~2-3 hours total (vs. 8-10 hours manual). The system processes in parallel, so batches complete faster. We load-test during Week 2 to ensure your infrastructure can handle peak volumes.

For Logistics Companies: Can it integrate with our TMS (Transport Management System)?

+

If your TMS has an API or accepts CSV imports, yes. Common systems (CargoWise, 3PL Central, Descartes) are supported. Phase 2 scopes the integration effort. If no API exists, we export to Excel/CSV for manual import.

What happens after you leave? Do we need ongoing support?

+

You own the source code and have full documentation. Most clients run independently. Tier 2 Support ($500/month) is optional for complex changes or AI retraining. Think of it as insurance, not dependency.

Can we add new document types ourselves after handover?

+

Yes. Simple documents (standard invoices, forms with consistent fields) can be added using the low-code interface. Complex documents (multi-page contracts, hand-written forms) may require AI retraining—that's where Tier 2 Support or a Re-engagement Sprint helps.

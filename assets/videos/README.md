# Hero Video Background

## Video Requirements

To add a video background to the hero section (similar to McKinsey's style), place your video files here:

- **File name**: `hero-background.mp4` (primary format)
- **File name**: `hero-background.webm` (fallback format for better browser support)

## Recommended Video Specifications

- **Format**: MP4 (H.264 codec) and WebM (VP9 codec)
- **Resolution**: 1920x1080 (Full HD) or higher
- **Aspect Ratio**: 16:9
- **Duration**: 10-30 seconds (will loop)
- **File Size**: Keep under 10MB for optimal loading
- **Content**: Abstract, subtle, professional - think:
  - Abstract tech/engineering visuals
  - Subtle data visualizations
  - Blue/navy color scheme to match brand
  - Slow, smooth motion
  - No distracting elements

## Video Style Inspiration

Similar to McKinsey's hero videos:
- Professional, premium feel
- Subtle motion
- Dark/blue color palette
- Abstract or conceptual visuals
- Not too busy - text should remain readable

## Fallback

If no video is provided, the hero section will use the solid navy blue background as defined in CSS.
